# Product Brief: SayDeck

- Status: Accepted
- Updated: 2026-07-20

## Product statement

SayDeckは、実際の場面で言いたかったことを日本語で入力し、AIによってAnkiで振り返りやすい英語表現へ変換・蓄積し、音声付きAPKGとして書き出す個人向けアプリである。

スケートボードの場面を主要な出発点にするが、日常会話、旅行、友人との雑談などにも同じ流れを使える。

## Problem

[事実] 学習者は基礎文法や暗記済みの英文を持っていても、実際に話したい内容をその場で英文に組み立てることが難しい。

[判断] SayDeckの責務はアプリ内で学習を完結させることではなく、現実の気づきを失わず、Ankiで復習できる高品質な教材へ変換することである。

## Goal

次の流れを、短時間かつモバイルブラウザから完結させる。

1. 言いたいことを日本語で入力し、ジャンルは必要な場合だけ選ぶ。
2. AIが日本語を読み取り、必須のシチュエーションタグ、ジャンル、難易度に合う基本ワードと例文を複数生成する。
3. 事前に定義したAnkiフィールドに沿って生成結果をDBへ保存する。
4. 保存済み表現を一覧し、ジャンル、シチュエーション、レベル、作成日時などで柔軟に絞り込む。
5. 選択した表現を米国英語音声付きAPKGとして書き出し、Ankiで復習する。

## Primary user and context

- Primary user: 本人（owner）
- Primary device: iOS browser
- Primary timing: スケートボード中、通勤中、日常で表現を思いついた直後
- Primary job: 思いつきを失わず、Ankiで復習可能な英語表現へ変換する

## Core experience

### INPUT

`言いたいこと`を日本語で入力する。シチュエーションは入力させず、AIが定義済みタグプールを優先して1〜3件の必須タグへ分類する。AI生成に失敗しても入力は失われない。ジャンルは`日常生活`、`スケートボード`、`その他`、`指定なし`から選び、`その他`だけ自由入力する。AIは難易度別の基本ワード、例文、訳、分類情報、Anki用フィールドを構造化して返し、DBへ保存する。

### LISTS

生成済みの表現を一覧する。ジャンル、シチュエーション、レベル、作成日時などで検索・絞り込み・グループ化し、内容を確認・修正・選択できる。

### EXPORT

選択した表現を、米国英語音声を内包するAnki互換`.apkg`として出力する。音声生成とWAV管理は内部処理であり、利用者に個別の「語句音声生成」「WAV生成」操作を要求しない。TSVは提供しない。

## Success measures

- 日本語の言いたいことだけを迷わず入力できる。
- AI失敗時も入力が残る。
- 1つの入力から難易度別の基本ワードと例文を比較・修正できる。
- 生成結果がAnkiフィールド契約に沿ってDBへ保存される。
- 保存済み表現を複数条件で一覧・絞り込み・選択できる。
- 生成したAPKGをAnkiへimportでき、例文と基本ワードの音声が米国英語として再生される。
- 再exportと再importで同じvariantが重複せず更新され、新しいカードは既存カードを上書きしない。
- 同じ難易度と主シチュエーションタグのカードが、同じAnki deck階層にまとまる。

## Product boundaries

- 初期版はowner中心で運用する。
- アプリ内学習、AI添削、練習履歴、復習間隔管理は扱わない。学習はAnkiの責務とする。
- 直接Anki同期、発音評価、日本語TTS、共同編集、公開deckは扱わない。
- 正式なexport形式は音声同梱APKGだけとし、TSVや音声ファイルの個別downloadは扱わない。
- 旧学習データは移行安全性のため直ちに削除しないが、現行UIには表示しない。
